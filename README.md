# Kirana Store Digitizer — Full Stack

A free, real-world web app to help small kirana store owners
manage inventory, billing, and daily sales digitally.

Built by **Aman Vashisath** | 

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Tailwind CSS, Recharts |
| Backend | Node.js, Express.js |
| Database | MySQL |

---

## Features

- Dashboard with live revenue, profit and low stock alerts
- Billing / POS — quick billing with printable receipt
- Inventory management — add, edit, delete products
- Sales history with filters
- Reports and analytics with charts
- Settings — store name and owner info

## How to Run Locally

### 1. Clone the repo
```bash
git clone https://github.com/amanvashisath/kirana-store-fullstack.git
cd kirana-store-fullstack
```

### 2. Start the backend
```bash
cd kirana-backend
npm install
npm run dev
```

### 3. Start the frontend
```bash
cd kirana-frontend
npm install
npm start
```

### 4. Set up the database
- Install MySQL
- Create database: `CREATE DATABASE kirana_db;`
- Create tables as per `/kirana-backend/server.js`
